# wai - college project
Gallery app, that allows users to save and browse images.

# Tech stack
- HTML, CSS
- JavaScript, JQuery, AJAX
- PHP
- MongoDB
- PHP Session API

# Features
- Registering users (with password encryption)
- Authentication of users saved in database
- Session mechanism
- Images upload
- Images serach using AJAX
- Displayed content based on user's upload
